# Front-end Style Guide

## Layout

Os designs foram criados para as seguintes larguras:

- Mobile: 375px
- Desktop: 1440px

> 💡 Estas são apenas as dimensões do design. Garanta que o conteúdo seja responsivo e atenda aos requisitos Web Content Accessibility Guidelines (WCAG) testando em toda a gama de tamanhos de tela, de 320px até telas grandes.

## Cores

### Primária

- Gold 500: hsl(31, 77%, 52%)
- Cyan 800: hsl(184, 100%, 22%)
- Green 950: hsl(179, 100%, 13%)

### Neutra

- Transparent white (paragraphs): hsla(0, 0%, 100%, 0.75)
- Gray 100 (background, headings, buttons): hsl(0, 0%, 95%)

## Tipografia

### Corpo do Texto

- Font size: 15px
